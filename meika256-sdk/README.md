# Meika-256 SDK

Native SDK for the Meika-256 cryptographic engine.

## Contents

- `include/` – C headers
- `lib/` – Import library (MSVC)
- `bin/` – Runtime DLL
- `examples/` – C usage example

## Supported Platforms

- Windows x64 (MSVC)

## Usage

See `examples/example.c`

Meika-256 is a secure encryption engine with both buffer and streaming APIs.
This SDK provides a stable C interface for use from native applications.
